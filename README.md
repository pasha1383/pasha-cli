# pasha CLI

ابزار CLI شخصی پارسا شادکام.

## نصب

### روش ۱ — npm
npm install -g pasha

### روش ۲ — مستقیم از GitHub
curl -fsSL https://raw.githubusercontent.com/pasha1383/pasha/main/install.sh | bash

## دستورات

pasha hello [name]
pasha hello پارسا --fancy
pasha info
pasha --version
pasha --help
